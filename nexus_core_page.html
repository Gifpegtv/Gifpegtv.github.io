Recursos Principais

Tudo que o NEXUS CORE faz

Do buffer inicial à reconexão silenciosa, cada frame é tratado com precisão cirúrgica.

🔧

Sutura Dinâmica

Reconecta streams ao vivo sem perda de sincronia. Busca reversa (10 offsets) e normal (10 offsets) para alinhar pacotes TS no byte 0x47 exato.

  • Busca primária em cache live
  • Fallback para cache backup
  • Timeout adaptativo (2s–8s)
  • Sliding window com alinhamento 0x47
🛡️

ContinuityFixer

Reescreve o campo Continuity Counter (CC) por PID em tempo real, eliminando glitches no MediaCodec e decodificadores Android.

  • Detecção de descontinuidade por PID
  • Resync automático pós-reconexão
  • Ignora null packets (0x1FFF)
  • Processamento packet-a-packet
⚡

Reconexão Adaptativa

5 estratégias de reconexão (S0–S4) com rotação de identidade (User-Agent + IP falso) a cada tentativa. Circuit breaker integrado.

  • Rotação de UA da pool VLC
  • IPs falsos dinâmicos (X-Forwarded-For)
  • Backoff exponencial com jitter
  • Circuit breaker com cooldown
💾

SingleTailCache

Sistema de cache live + backup que promove dados uma única vez por reconexão, garantindo overlap perfeito entre conexões antigas e novas.

  • Cache live em memória
  • Backup promovido automaticamente
  • Tamanho dinâmico baseado em bitrate
  • Alinhamento em 188 bytes
📊

Controle de Latência

Ajusta dinamicamente o tamanho dos chunks de leitura baseado na latência média da rede, balanceando velocidade e estabilidade.

  • Janela de 20 amostras
  • Target: 180ms por chunk
  • Range: 1.5KB – 256KB
  • Ajuste proporcional ±10%–25%
🎯

Null-Packet Pump

Evita EOF prematuro no Kodi injetando bursts de 64 null packets a cada 80ms quando a fila do cliente esvazia durante reconexão.

  • 64 packets TS null (0x1FFF)
  • Intervalo: 80ms
  • Anti-travamento do player
  • Ativado apenas em stall

Proxy HLS

Adaptive HLS Proxy

Reescreve playlists M3U8 em tempo real, analisa métricas de rede e reconecta proativamente antes que o buffer zere.

🔄

Reescrita de Playlist

Intercepta a playlist M3U8 e substitui todas as URLs de segmentos e chaves por endpoints locais do proxy, mantendo tokens e query strings intactos.

📡

StreamAnalyzer

Monitora latência, jitter e velocidade de download em tempo real. Dispara reconexão proativa quando detecta risco de buffer underrun.

  • Média móvel de 5 amostras
  • Detecção de alta latência (>2s)
  • Detecção de jitter (>1.2s)
  • Risco de bitrate comparativo
⚙️

Perfis de Player

Emula headers de players reais (ExoPlayer, VLC, Kodi) para máxima compatibilidade com servidores de streaming que filtram por User-Agent.

  • ExoPlayer/Lavf60.3.100
  • VLC/3.0.20 Vetinari
  • Lavf/60.3.100 (Kodi)
  • Redirecionamentos 301-308
💿

LRU Cache de Segmentos

Mantém os últimos 5 segmentos em cache de memória. Se o Kodi solicitar um segmento recente, a resposta é instantânea, sem novo download.

Detecção Inteligente de URL

O addon detecta automaticamente o tipo de stream baseado no path da URL, sem necessidade de configuração manual.

# Tipos suportados .ts → ts_ext .m3u8 → hls /123456 → ts_noext /AbCd/123 → ts_noext
🧠

Conversão Automática

URLs são convertidas automaticamente entre TS e HLS conforme o proxy escolhido, preservando query strings e tokens de autenticação.

# TS sem extensão → HLS /live/AbCd/123 → /live/AbCd/123.m3u8 # HLS → TS /stream.m3u8 → /stream.ts
